# SOC Playbooks

> A collection of professionally documented Security Operations Center (SOC) incident response playbooks based on industry best practices.

---

## About

This repository contains structured incident response playbooks for common security events encountered in enterprise environments.

The goal of this project is to document repeatable investigation and response procedures that align with real-world SOC workflows.
