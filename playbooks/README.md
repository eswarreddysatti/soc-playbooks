# Playbooks

This directory contains incident response playbooks covering common security incidents encountered in enterprise Security Operations Centers (SOCs).

## Available Playbooks

| Playbook               | Status |
| ---------------------- | ------ |
| Phishing Investigation | ✅      |
| Malware Investigation  | ✅      |
| Suspicious Login       | ✅      |

Each playbook follows a consistent structure:

* Overview
* Severity Classification
* MITRE ATT&CK Mapping
* Detection
* Triage
* Investigation
* Containment
* Recovery
* Lessons Learned
* References
